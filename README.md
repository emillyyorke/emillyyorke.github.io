## 📋 Sobre o Projeto

Este portfólio foi desenvolvido como uma vitrine profissional e acadêmica, reunindo em um único lugar todas as informações relevantes sobre minha formação, habilidades técnicas, experiências de trabalho, projetos desenvolvidos e produção científica.

O design segue uma proposta **clean e moderna**, com uma paleta de cores em tons pastéis (lilás, azul claro e rosa), elementos visuais que remetem à computação (padrão de circuitos no fundo) e animações suaves de scroll, criando uma experiência agradável e memorável para o visitante.

---

## ✨ Funcionalidades

- **Design Responsivo** — layout totalmente adaptável para desktop, tablet e celular
- **Navegação Fixa** — barra de navegação com efeito glassmorphism (blur) que acompanha o scroll
- **Scroll Reveal** — animações de entrada dos elementos conforme o usuário rola a página
- **Menu Mobile** — menu hambúrguer funcional para telas menores
- **Destaque Ativo na Navbar** — o link da seção visível é destacado automaticamente
- **Badges Flutuantes** — elementos animados ao redor da foto com efeito de flutuação
- **Cards Interativos** — hover effects nos cards de projetos, habilidades e interesses
- **Links Externos** — integração direta com GitHub, LinkedIn, Lattes e e-mail

## 🛠️ Tecnologias e Ferramentas

### Linguagens
| Tecnologia | Utilização |
|---|---|
| **HTML5** | Estrutura semântica de todo o conteúdo do site |
| **CSS3** | Estilização, layout (Grid/Flexbox), animações e responsividade |
| **JavaScript** | Interatividade, scroll reveal, menu mobile e navbar dinâmica |

### Recursos e Técnicas de CSS
- **CSS Custom Properties (variáveis)** — paleta de cores e tokens de design centralizados
- **CSS Grid e Flexbox** — layouts responsivos e alinhamento dos componentes
- **Keyframe Animations** — animações de fade, wave e float
- **Backdrop Filter (Glassmorphism)** — efeito de desfoque na navegação
- **Gradientes Lineares** — banners dos projetos e fundo da foto
- **Media Queries** — breakpoints para responsividade mobile
- **SVG Inline** — padrão de circuito no fundo e ícone placeholder

### Recursos de JavaScript
- **Intersection Observer API** — detecção de elementos visíveis para animações de scroll reveal
- **Event Listeners** — controle do menu mobile e efeitos de scroll
- **classList API** — toggle de classes para estados ativos e animações
- **DOM Manipulation** — animação programática da saudação no hero

### Fontes (Google Fonts)
- **DM Serif Display** — títulos e destaques (serif elegante)
- **Nunito** — texto corrido e interface (sans-serif arredondada)
- **Fira Code** — disponível para elementos de código (monospace)

### Hospedagem
- **GitHub Pages** — deploy automático e gratuito a cada push no repositório

## 🚀 Como Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/emillyyorke/emillyyorke.github.io.git
   ```

2. Acesse a pasta:
   ```bash
   cd emillyyorke.github.io
   ```

3. Abra o `index.html` no navegador:
   ```bash
   # No Linux/Mac
   open index.html

   # No Windows
   start index.html
   ```

Não é necessário instalar dependências — o projeto é 100% vanilla (HTML, CSS e JS puros).

## 📝 Como Personalizar

Para adaptar este portfólio ao seu perfil, edite os seguintes pontos:

- **`index.html`** — textos, links, projetos, publicações e informações pessoais
- **`style.css`** — cores (altere as variáveis no `:root`), fontes e espaçamentos
- **`foto.jpg`** — substitua pela sua foto de perfil
- Os locais editáveis estão marcados com comentários `<!-- ✏️ ... -->` no HTML
